# 3D Letter A (OpenGL)

## Build

```bash
g++ main.cpp -lGL -lGLU -lglut -o letterA
```

You need an OpenGL + GLUT toolkit (e.g., freeglut or glut) installed for the headers and libraries.

## Controls

- `W/S` – rotate around X
- `A/D` – rotate around Y
- `Q/E` – rotate around Z
- `I/K` – move up/down
- `J/L` – move left/right
- `U/O` – move forward/backward
- `+/−` – increase/decrease uniform scale
- `Esc` – exit
